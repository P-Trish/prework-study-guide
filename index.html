Prework Study Guide

HTML

  • HTML stands for Hypertext Markup Language.
  • The head element contains metadata, information about the webpage.
  • The body element represents the visible content shown to the user.
  • The child elements are elements that are nested inside other elements.

CSS

  • A margin indicates how much space we want around the oustdie of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A box shadow adds shadow effects around an element's frame.
  • Assigning colors in CSS can be done by semantic references such as "Gray" or "Blue" or referring to a color by its hexadecimal code or hex #.
  • Do Not Repeat Yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained.
  • The period preceding "card" designates this as a class selector. Any HTML element assigned to the class, "card" will gain these property assignments from the CSS.

Git

  • Git is a version control system that manages and keeps track of each version of of a project
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
