Prework Study Guide

HTML

  • HTML stands for Hypertext Markup Language.
  • The head element contains metadata, information about the webpage.
  • The body element represents the visible content shown to the user.
  • The child elements are elements that are nested inside other elements.

CSS

  • A margin indicates how much space we want around the oustdie of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A box shadow adds shadow effects around an element's frame.
  • Assigning colors in CSS can be done by semantic references such as "Gray" or "Blue" or referring to a color by its hexadecimal code or hex #.
  • Do Not Repeat Yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained.
  • The period preceding "card" designates this as a class selector. Any HTML element assigned to the class, "card" will gain these property assignments from the CSS.

Git

  • Git is a version control system that manages and keeps track of each version of of a project
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Best Practice: Always write a semicolon after a statement to reduce chances of bugs getting in the code

  • A string is surrounded by quotation marks

  • An array is a single variable used to hold a group of related data
  • Conditional statements interrupt control flow so that different code executes depending on a condition that evaluates as truthy ("if" or "else if")
  • An "if" statement introduces decision-making in code so that code will only execute if the statement meets a condition or is true
  • A "loop" uses predictable pattern of indices to perform a tas on all the items in an arrayby allowing a single code block to be executed over and over
  • "For" loops contain three importatnt statements: 1. keyword 'for', 2. three-part conditional statement, 3. code block (console.log) statement
  • A function is a set of instructions that tells the computer how to perform a certain task
  • 2 Requirements for a JavaScript function: 1. Define the function with a function declaration, 2. Call the function for it to execute
  • Important Note: Always include parentheses whenever you write a function
